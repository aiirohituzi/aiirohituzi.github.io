react-bootstrap 사용 시 style 덮어쓰는 방법

기본적으로 bootstrap에 정의된 css를 따라가기에 style을 직접 덮어씌워야 변경이 가능한데

stack overflow 를 통해 그 방법을 찾을 수 있었다.
